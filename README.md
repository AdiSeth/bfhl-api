# BFHL API

A simple Node.js + Express API deployed on **Vercel**.  
This API takes an array of mixed inputs (numbers, alphabets, special characters) and returns:

- Even numbers  
- Odd numbers  
- Alphabets (uppercase)  
- Special characters  
- Sum of all numbers  
- A concatenated string (custom transformation)
